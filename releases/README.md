# AI Email App Releases

## Latest Release - v1.0.0

### Download
- **File**: `ai-email-app-1.0.0-jar-with-dependencies.jar`
- **Size**: ~65 MB
- **Java Version**: Requires Java 11 or higher

### What's Included

This JAR file contains everything you need to run the AI Email App:
- Complete Java application
- All dependencies bundled
- No additional downloads required

### Quick Start

1. **Download** the JAR file
2. **Run** it with:
   ```bash
   java -jar ai-email-app-1.0.0-jar-with-dependencies.jar
   ```
3. **Configure** on first launch:
   - Optimum email credentials
   - Google Gemini API key

### Features

✅ **AI-Powered Email** - Uses Google Gemini for intelligent email composition  
✅ **Voice Control** - Talk to the AI to control the app  
✅ **Simple Interface** - Large fonts and buttons designed for elderly users  
✅ **Auto-Startup** - Launches automatically on system boot  
✅ **Desktop Icon** - Creates desktop shortcut on first run  
✅ **Persistent Login** - Stays logged in to your email account  
✅ **Optimum Email** - Direct integration with Optimum email accounts  

### System Requirements

- **Operating System**: Windows 10+, macOS 10.14+, or Linux (any modern distribution)
- **Java**: Version 11 or higher
- **Memory**: 512 MB RAM minimum, 1 GB recommended
- **Internet**: Active connection required for email and AI features
